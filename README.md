# Bynaryscode

BynarysCode est une bibliothèque de code utilitaire dont je me sert pour mes autres projets.

Inclus notamment :
  - un package maths avec des outils mathématiques tels que les vecteurs, des méthodes utilitaires dans la classe MathUtil
  - un package path avec des outils de pathfinding
  - un package swing avec du code utilitaire pour les interface graphiques utilisant Swing
  - des hors catégories : outils de log, structures de données, exceptions, ...
